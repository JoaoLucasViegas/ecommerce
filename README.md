<h1> Ecommerce </h1>
<h3> Desenvolvido como exercício de fixação de conteúdos aprendidos durante o programa educacional "Jovem Tech" no período de Julho de 2026! </h3>
<br>
<h4> O programa consiste em acessar uma página de produtos, realizar adição e remoção de produtos de um carrinho de compras virtual, e realizar fim do ciclo de compra efetuando o pagamento. </h4>
<h4> Através de uma interface de usuário, o programa permite realizar gerenciamento de itens, adicionando ou removendo e definindo quantidade de itens de forma unitária</h4>
<h4> Fazendo também a realização de desconto de créditos na carteira virtual do usuário após serem realizadas com sucesso compras dos produtos oferecidos no catálogo da página inicial. </h4>
